## Getting Started

### Prerequisites

- Node.js 18+
- Samsung SmartThings Personal Access Token

### Installation

```bash
npm install
npm run dev
```

## SmartThings Setup

1. Go to [https://account.smartthings.com/tokens](https://account.smartthings.com/tokens)
2. Generate a new Personal Access Token with device permissions
3. Copy the token (it's only shown once!)
4. Paste it in the app settings
